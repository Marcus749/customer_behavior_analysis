# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow, starting from raw data and ending with interactive business insights. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation in Power BI, and presentation of findings through a professional report and presentation.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

## How to Run

1. Clone the repository.
2. Install the required Python libraries.
3. Run the notebooks in the following order:
   - Data Loading
   - EDA
   - Data Cleaning
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Execute the SQL scripts.
6. Open the `.pbix` file in Microsoft Power BI Desktop.
7. Explore the dashboard and review the report and presentation.
